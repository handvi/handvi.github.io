About Me

I'm Faisal Hanafi, a Cyber Security Researcher and software developer specializing in ethical hacking, AI engineering, and Web3 development. My mission is to build secure systems while identifying and mitigating vulnerabilities in existing ones.

With expertise in penetration testing, red teaming, and vulnerability research, I approach security from an attacker's perspective to strengthen defenses. My development skills focus on creating security tools with Rust, building AI-powered security solutions with Python, and developing smart contracts with Solidity.

As an AI Engineer, I develop machine learning models for threat detection, anomaly identification, and automated security analysis. My Web3 work focuses on secure smart contract development and auditing for decentralized applications.

My Contributions

I've actively contributed to making programming education accessible by translating freeCodeCamp's extensive learning materials into Indonesian. This work directly supports a global community of aspiring developers, empowering them with knowledge in their native language.

I have experience in developing custom penetration testing tools for Kali Linux using Python. This involves creating specialized scripts and utilities to automate security assessments and identify vulnerabilities, enhancing the capabilities of ethical hackers and security professionals.
